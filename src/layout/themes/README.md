# Themes

Theme names you can set in config as `theme = "..."` in `ublx.toml` or `.ublx.toml`. Use the theme selector (Ctrl+t) in the TUI to preview and save.

The picker lists **Dark** and **Light** section headers; under each, themes are sorted **alphabetically (A–Z)** by display name. The tables below use the same order.

## Allowable values

### Dark

| Theme name             | Description                                                                                                   |
| ---------------------- | ------------------------------------------------------------------------------------------------------------- |
| **Archival Simulacra** | True black page, neon green phosphor body text, bright green focus and tabs, dim emerald hints - matrix vibe. |
| **Babel Blend**        | Deep navy page, warm parchment body text, orange focus/search vs brick-red active tabs and coral hints.       |
| **Burning Glyph**      | Maroon-black page, pale buttery text, coral-red focus and search, amber brand.                                |
| **Frozen Phrase**      | Nordic blue-gray page, snow text, frost-blue focus, ice-pale active tabs, sea-glass search, misty hints.      |
| **Garden Unseen**      | Deep forest-green page, peach-cream text, mint focus and search, olive-brown brand.                           |
| **Golden Delirium**    | Olive-black page, soft mint-cream text, yellow-lime focus and search, rust-brown brand.                       |
| **Oblivion Ink**       | **Default.** Deep navy page, pale aqua text, bright cyan focus and search, magenta hints and brand.           |
| **Purple Haze**        | Near-black violet page, lavender-white text, magenta-pink focus and search, violet brand.                     |
| **Resin Record**       | Amber-terminal dark theme: near-black page, phosphor amber text, warm amber focus/search, compact CRT feel.   |
| **Shadow Index**       | True black page, cool off-white body text, medium-gray focus, hints with a slight blue-gray cast.             |
| **Tangerine Memory**   | Burnt umber page, light honey-cream text, peach-gold focus and search, dusty rose brand.                      |

### Light

| Theme name        | Description                                                                                                |
| ----------------- | ---------------------------------------------------------------------------------------------------------- |
| **Asterion Code** | Cool blue-gray stone page, blue-teal body text/pool-tile chrome, warm clay hints.                          |
| **Barley Bound**  | Gruvbox-style buttercream page, warm dark body text, teal focus, olive search, orange brand, stone hints.  |
| **Cold Trace**    | Nord-style snow page, polar-night body text, frost-blue focus/search, slate tabs, icy brand, muted hints.  |
| **Cryptic Chai**  | Tea-stained tan parchment, dark chocolate body text, copper-brown focus and tabs, olive-tan hints.         |
| **Faded Echo**    | Dusty sepia paper theme: warm parchment background, book-ink text, copper-brown accents, archival calm.    |
| **Infinite Rose** | Very pale steel-gray page, rose-gold body text/chrome, dusty mauve hints.                                  |
| **Obdurate Noon** | Solarized-light inspired parchment page, muted blue-gray text, cyan focus, blue search/tabs, violet hints. |
| **Ochre Thread**  | Pale warm sand page, burnt-orange body text, copper focus and rust tabs, blue-gray hints.                  |
| **Pale Mirror**   | Cool frosted blue-lilac page, dark plum body text, purple tab/focus chrome, rose-mauve hints.              |
| **Parched Page**  | Warm cream manuscript page, forest-green body text and green tab chrome, amber-brown hints.                |
| **Silent Sheet**  | White page, black text, charcoal focus, slate-blue active tabs, warm gray hints.                           |

Omit `theme` or set `theme = "default"` to use **Oblivion Ink**.
